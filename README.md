# 🎮 Tic Tac Toe Game (Streamlit)

A simple but interactive Tic Tac Toe game built using **Python, NumPy, and Streamlit**.

---

## 🌐 Live Demo
👉 https://tic-tac-toe-app-hppjy6gv2kdi9kg4oo9og9.streamlit.app/

---

## ✨ Features
- 🎯 Click-based gameplay (no typing needed)
- ❌ X and ⭕ O turn system
- 🧠 Automatic winner detection
- 🤝 Draw detection
- 🔄 Reset game button
- 💡 Simple and clean UI

---

## 🛠️ Tech Stack
- Python
- Streamlit
- NumPy

---

## 🚀 How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
